Use

  1. Drag this to your bookmarks bar: BttrWrtr

  2. Click the bookmark on any page you want to check for weasel words, lexical illusions (duplicate words), and passive voice.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/lukegotszling/bttrwrtr

Or with Mercurial using the hg-git extension by running: 

$ hg clone git://github.com/lukegotszling/bttrwrtr.git
$ hg update master
